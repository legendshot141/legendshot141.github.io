Guess the Color


This is a side project that I have worked on which is a game called Guess the color. The user must guess the color that will display by writing their choice in the textbox and then clicking the submit button. The colors are either red, blue, yellow, or green. If the user guess correctly they win, if not they lose. The languages that I used was HTML & CSS, and JavaScript.
